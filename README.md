# 🛡️ LangChain PR Gatekeeper

Think of this as an AI assistant for your code! **PR Gatekeeper** uses LangChain to automatically review your pull requests, catch mistakes, and give developers helpful feedback so you can merge code faster and safer.

## ✨ What it does

* **Reads your code:** Automatically checks any new code in a Pull Request (PR).
* **Finds bugs:** Spots errors and logical mistakes before they get merged into the main project.
* **Suggests fixes:** Gives you clear, easy-to-understand tips on how to improve the code.
* **Saves time:** Does the heavy lifting of code review so developers can focus on building new features!

## 🛠️ Built With

* **LangChain:** Powers the AI logic to connect everything together.
* **Large Language Models (LLMs):** The "brain" that reads and understands your code changes.

## 🚀 How it works

1. A developer opens a new Pull Request on GitHub.
2. The **PR Gatekeeper** wakes up and reads exactly what changed.
3. The AI thinks about the code and looks for any hidden bugs or messy formatting.
4. It leaves a friendly, helpful comment right on your PR with its feedback!
